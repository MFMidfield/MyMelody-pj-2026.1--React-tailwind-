My Melody Website
โปรเจกต์นี้เป็นเว็บไซต์ธีม My Melody ที่สร้างขึ้นเพื่อฝึกพัฒนาเว็บด้วย React + Tailwind CSS
โดยนี่เป็น ครั้งแรก ที่ใช้ React และ Tailwind แบบจริงจังในการทำเว็บทั้งโปรเจกต์
เว็บไซต์นี้เน้นการออกแบบให้ดูน่ารัก สบายตา และเป็นมิตรกับผู้ใช้

Home
    เป็นหน้าต้อนรับของเว็บไซต์ ใช้เพื่อแนะนำโปรเจกต์และตัวผู้พัฒนา ในหน้านี้จะมีข้อความที่บอกอย่างชัดเจนว่า โปรเจกต์นี้เขียนโค้ดเองประมาณ 90% และใช้ AI ช่วยประมาณ 10%

    เป้าหมายของหน้านี้คือ
        แสดงความตั้งใจและความพยายามในการเขียนโค้ดด้วยตัวเอง
        ใช้ AI เป็นแค่เครื่องมือช่วย ไม่ใช่ตัวเขียนโค้ดแทน
        ทำให้ผลงานดูโปร่งใสและน่าเชื่อถือ

About Section
    ส่วนนี้เป็นส่วนหลักของเว็บไซต์ ใช้แนะนำตัวละคร My Melody
    โครงสร้างของส่วนนี้แบ่งออกเป็น 2 ส่วนหลัก

    1. หัวข้อและรายการ (Main Info)
        จะมีหัวข้อใหญ่ “My Melody” และรายการย่อย 4 ข้อ เพื่ออธิบายลักษณะเด่นของตัวละคร เช่น บุคลิก สไตล์โทนสี ความรู้สึกที่สื่อออกมา ส่วนนี้ช่วยให้ผู้ใช้งานเข้าใจ My Melody ได้อย่างรวดเร็ว

    2. Cards (การ์ดข้อมูล)
        ด้านล่างจะมี การ์ดทั้งหมด 3 ใบ แต่ละการ์ดประกอบด้วย หัวข้อ, ข้อความอธิบาย
        การ์ดใช้เพื่อแยกข้อมูลออกเป็นหมวดหมู่ เช่น My Melody คือใคร, สไตล์ของเธอ, เหตุผลที่ผู้คนชอบเธอ
        การใช้ Card ช่วยให้ UI ดูเป็นระเบียบ อ่านง่าย และดูเป็น modern web

Tech Stack
    โปรเจกต์นี้ใช้เทคโนโลยีหลักคือ
        React – สำหรับสร้าง UI แบบ Component
        Tailwind CSS – สำหรับจัดการสไตล์และเลย์เอาต์


โปรเจกต์นี้เป็นการฝึกใช้งาน React + Tailwind ครั้งแรก เพื่อพัฒนาทักษะการทำ Frontend ให้ดีขึ้น
